Регистрация на сайте Риот Валорант

Перейдите по ссылке https://playvalorant.com/ru-ru/. Далее создайте свой Riot ID по инструкции ниже.

  1. Нажмите на кнопку «Играть».

  2. Если уже есть аккаунт, тогда нажмите на вторую кнопку «Авторизоваться», далее введите свои регистрационные данные — логин и пароль.

  3. Если аккаунта еще нет, то без него скачать Valorant хоть и получится (кнопки в начале страницы), но играть в дальнейшем не получится. Клацните на первую кнопку «Создать» и, следуя шаг за шагом, введите необходимые данные. Задайте уникальный логин, надежный пароль, состоящий из 8 символов минимум и наличием больших букв и цифр.

  4. После регистрации вас перекинет на страницу, где будет кнопка для скачивания последней версии игры. В это же время произойдет автоматическая авторизация в аккаунт. Кликните на «Загрузить», система предложит сохранить установочный файл на компьютер. Выберите удобное место для сохранения и начните загрузку. Письмо с ссылкой для загрузки файла также придет на электронную почту, которая указывалась при регистрации.
